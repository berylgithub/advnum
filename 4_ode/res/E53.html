All of the meshes are generated using Triangle software.
Example of mesh generation command: "triangle -p circle.poly", generates circle mesh from "circle.poly" file.
Example of mesh refinement command: "triangle -rpa0.5 circle.poly", refines "circle.poly" by 0.5 refinement parameter.
Example of image generation command: "showme circle.poly".
